WEEK 3: WAITING & SYNCHRONIZATION
---------------------------------

Auto-waiting (Built-in):
- Playwright automatically waits for elements to be actionable
- Waits for element to be visible, stable, enabled
- Default timeout: 30 seconds

Explicit Waiting:
[ ] page.wait_for_selector(selector)
[ ] page.wait_for_load_state("networkidle")
[ ] page.wait_for_load_state("domcontentloaded")
[ ] page.wait_for_load_state("load")
[ ] page.wait_for_timeout(ms) - Use sparingly!
[ ] page.wait_for_function(js_function)
[ ] page.wait_for_url(url_pattern)
[ ] page.wait_for_response(url_pattern)
[ ] page.wait_for_request(url_pattern)

Locator Waiting:
[ ] locator.wait_for() - Wait for element
[ ] expect(locator).to_be_visible()
[ ] expect(locator).to_have_text()

Example:
    # Wait for element to appear
    page.wait_for_selector(".results-loaded")
    
    # Wait for network to be idle
    page.goto(url)
    page.wait_for_load_state("networkidle")
    
    # Wait for specific response
    with page.expect_response("**/api/data") as response_info:
        page.click("#load-data")
    response = response_info.value

ASYNC PLAYWRIGHT
----------------
[ ] Understanding asyncio basics
[ ] Async context managers
[ ] Concurrent page operations
[ ] Performance benefits

Async Example:
    import asyncio
    from playwright.async_api import async_playwright

    async def main():
        async with async_playwright() as p:
            browser = await p.chromium.launch()
            page = await browser.new_page()
            await page.goto("https://example.com")
            title = await page.title()
            print(title)
            await browser.close()

    asyncio.run(main())

Concurrent Scraping:
    async def scrape_page(browser, url):
        page = await browser.new_page()
        await page.goto(url)
        title = await page.title()
        await page.close()
        return title

    async def main():
        async with async_playwright() as p:
            browser = await p.chromium.launch()
            urls = ["https://example1.com", "https://example2.com"]
            tasks = [scrape_page(browser, url) for url in urls]
            results = await asyncio.gather(*tasks)
            print(results)
            await browser.close()

    while True:
        # Scroll to bottom
        page.evaluate("window.scrollTo(0, document.body.scrollHeight)")
        page.wait_for_timeout(2000)  # Wait for content
        
        # Check if we've reached the end
        new_height = page.evaluate("document.body.scrollHeight")
        if new_height == previous_height:
            break
        previous_height = new_height

COMBINING TOOLS
---------------
Decision Matrix:
    
    Use Requests + BeautifulSoup when:
    - Static HTML pages
    - No JavaScript rendering needed
    - Simple data extraction
    - High-speed scraping needed
    
    Use Playwright when:
    - JavaScript-rendered content
    - User interactions required
    - Login/authentication needed
    - Complex navigation flows
    - Anti-bot measures present

Hybrid Approach:
    # Use Playwright to get rendered HTML, then parse with BeautifulSoup
    from playwright.sync_api import sync_playwright
    from bs4 import BeautifulSoup
    
    with sync_playwright() as p:
        browser = p.chromium.launch()
        page = browser.new_page()
        page.goto("https://example.com")
        page.wait_for_load_state("networkidle")
        
        html = page.content()
        soup = BeautifulSoup(html, 'lxml')
        
        # Fast parsing with BeautifulSoup
        items = soup.select(".item")
        browser.close()

ERROR HANDLING & RELIABILITY
----------------------------
[ ] Try/except blocks
[ ] Retry mechanisms
[ ] Timeout handling
[ ] Graceful degradation

Robust Scraping Example:
    from playwright.sync_api import TimeoutError
    import time
    
    def scrape_with_retry(page, url, max_retries=3):
        for attempt in range(max_retries):
            try:
                page.goto(url, timeout=30000)
                page.wait_for_selector(".content", timeout=10000)
                return page.content()
            except TimeoutError:
                print(f"Attempt {attempt + 1} failed, retrying...")
                time.sleep(2 ** attempt)  # Exponential backoff
        
        raise Exception(f"Failed to scrape {url} after {max_retries} attempts")
