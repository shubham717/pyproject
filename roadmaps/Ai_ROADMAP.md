# AI Learning Roadmap (Python-First, Beginner)

This roadmap is designed for a Python beginner who wants to learn AI step by step.
Timeline is flexible; adjust based on your weekly time budget.

## Phase 0: Python Foundations (2-4 weeks)
Goal: Be comfortable writing small programs without help.

Topics:
- Syntax, variables, data types
- Control flow (if/elif/else, loops)
- Functions (arguments, return values)
- Data structures (list, dict, set, tuple)
- Strings, slicing, common methods
- File I/O
- Exceptions
- Basic OOP (classes, methods, attributes)

Practice:
- 20-30 short problems (loops, strings, lists)
- Write a small script that reads a file and outputs a summary

Deliverables:
- A folder of 20+ solved Python exercises
- A small CLI script (e.g., text analyzer)

## Phase 1: Data Skills for ML (2-4 weeks)
Goal: Load, clean, and explore data with Python.

Topics:
- NumPy arrays, vectorized operations
- Pandas DataFrames (filtering, joins, groupby)
- Handling missing values
- Simple visualization (Matplotlib, Seaborn)

Practice:
- Load a CSV and answer 10 questions from it
- Clean a messy dataset and document your steps

Deliverables:
- One Jupyter notebook or script that cleans and explores a dataset
- 3-5 charts with short insights

## Phase 2: Math Essentials (Ongoing, 2-6 weeks in parallel)
Goal: Understand the math used in ML models.

Topics:
- Linear algebra: vectors, matrices, dot product, norms
- Probability: distributions, Bayes rule
- Statistics: mean, variance, standard deviation
- Calculus: derivatives, gradients (only basics)

Practice:
- Compute dot products and matrix multiplications in NumPy
- Visualize distributions with Python

Deliverables:
- A short math notes file with examples
- A few small Python scripts that implement formulas

## Phase 3: Core Machine Learning (4-8 weeks)
Goal: Train and evaluate classic ML models.

Topics:
- Train/validation/test split
- Overfitting, underfitting, bias vs variance
- Regression: linear regression, regularization
- Classification: logistic regression, kNN, SVM
- Trees: decision trees, random forests
- Unsupervised: k-means, PCA
- Evaluation: accuracy, precision, recall, F1, ROC-AUC
- Model selection: cross-validation

Tools:
- scikit-learn

Practice:
- Build a classifier on a simple dataset (e.g., Titanic)
- Compare 3 models on the same dataset

Deliverables:
- 2 ML mini-projects with README
- A summary report comparing model performance

## Phase 4: Deep Learning Basics (4-8 weeks)
Goal: Understand and train neural networks.

Topics:
- Neural networks and backpropagation
- Activation functions, loss functions
- Optimizers (SGD, Adam)
- MLPs for tabular data
- CNNs for images
- RNNs or Transformers for text (intro level)

Tools:
- PyTorch (recommended)

Practice:
- MNIST digit classifier
- Simple text classifier

Deliverables:
- 1 image project + 1 text project with training curves

## Phase 5: Projects + Deployment (Ongoing)
Goal: Build a portfolio and share your work.

Topics:
- End-to-end ML pipeline
- Basic deployment with Flask or FastAPI
- Model versioning and experiment tracking (optional)

Practice:
- One end-to-end project (data -> model -> report)
- Optional: simple API that returns predictions

Deliverables:
- 3-5 portfolio projects
- A short README for each project

## 8-Week Starter Plan (Example)
Weeks 1-2:
- Python basics + exercises

Weeks 3-4:
- NumPy/Pandas + 2 data mini-projects

Weeks 5-6:
- ML basics with scikit-learn

Weeks 7-8:
- ML project + report
